Esquema de conexión DHT22-Arduino

20 sensores repartidos en dos microcontroladoras, utilizando hilos de cables UTP para las conexiones.

  • Sensor: DHT-22
  • Placa: Arduino Uno
  • Resistencia: 10K
  • Conexiones: Pin Digital, 5V, GND

Código en Arduino

El código se repite por cada sensor.

El formato de impresión de elementos está pensado para interactuar con el programa en Python de la Raspberry Pi.

#include <DHT.h>
#define DHTTYPE DHT22
const int DHTPin1 = 2;
DHT dht1(DHTPin1, DHTTYPE);
void setup() {
  Serial.begin(9600);
  dht1.begin();
}

void loop() {
  float h = dht1.readHumidity();
  float t = dht1.readTemperature();
  Serial.print("{\"humidity\":");
  Serial.print(h);
  Serial.print(",");
  Serial.print("\"temperature\":");
  Serial.print(t);
  Serial.print(",");
  Serial.print("\"id\":");
  Serial.print(1);
  Serial.println("}\n");
  delay(300000);
}

Código en Raspberry Pi para conectar con Arduino

Leemos el serial y guardamos la información en la base de datos.

import serial
import json
import MySQLdb

arduino = serial.Serial('/dev/ttyACM0',9600)

while True:
    character= arduino.readline()
    MyJson = character
    db = MySQLdb.connect("localhost","root","password","basededatos")
    if character != '\n':
        try:
            data=json.loads(character)
            print data

            curs = db.cursor()
            curs.execute("INSERT INTO registro(fecha,id,hum,temp)values(now(),"+str(data['id'])+','+str(data['humidity'])+","+str(data['temperature'])+")")
            db.commit()
