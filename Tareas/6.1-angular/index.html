Paso a paso

Instalación, ejecución y prueba del repositorio Spring Boot en el módulo 6

En este paso a paso veremos cómo se ha resuelto la actividad 6.1 del curso ANGULAR - IONIC. APLICACIONES FULL STACK CON SPRING BOOT Y JPA.

1. Actualización del repositorio a la rama main

Actualizamos el repositorio de Spring Boot (Módulo 2), y reseteamos a la última rama del desarrollo (o main).

2. Ejecución del proyecto Spring Boot

Ejecutamos el proyecto Spring Boot, para asegurarnos de que nuestro backend está funcionando.

3. Instalación de Angular

Instalamos Angular (habiendo instalado npm antes) mediante “npm -g install @angular/cli”.

4. Instalación de las librerías del proyecto

Instalamos todas las librerías del proyecto Angular desde Visual Studio Code mediante “npm install”.

5. Ejecución del proyecto Angular

Ejecutamos el proyecto Angular, mediante “npm serve -o”

6. Comprobación de la ejecución de la aplicación

Entramos en localhost:4200 para asegurarnos de que la aplicación está funcionando.

7. Prueba de inicio de sesión

Probamos que podemos iniciar sesión y acceder a los mensajes.
